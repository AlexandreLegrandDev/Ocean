# Découvrez l'Océan

Site one-page permettant de découvrir la bibliothèque GSAP et l'accessibilité 

## Objectifs du site

- lier accessibilité et design graphique
- proposer une expérience interactive et immersive
- créer une page éducative et intéressante

## Technologies utilisées

- HTML & CSS
- SCSS
- JavaScript (+ React.js)

## Thème 

Ce site permet de découvrir différentes espèces marines selon le niveau de profondeur. Basé sur l'océan, il reprend le principe d'un site one-page qui révèle les profondeurs à mesure que l'on scroll vers le bas. On retrouve également une section sur la droite de la page qui donne de nombreuses informations sur le niveau de profondeur.
