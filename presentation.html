Un site éducatif et immersif sur les profondeurs océaniques. Le scroll devient une plongée progressive de la surface jusqu'aux abysses.

Stack technique

  • HTML5 sémantique
  • SCSS (compilé)
  • JavaScript Vanilla
  • GSAP + ScrollTrigger

Personne 2

Fonctionnalités techniques

🐟 Nuée de poissons

Animations GSAP par zone, images PNG découpées, vitesse variable selon la profondeur et la taille.

🌊 Double Canvas

Vagues animées dans le hero, bioluminescence dans les abysses — désactivées si prefers-reduced-motion.

📊 HUD dynamique

Panneau latéral sticky : température, pression, luminosité, sonar — mis à jour via ScrollTrigger.

🎴 Modal espèce

Overlay avec image locale, nom latin, profondeur et lien Wikipedia. Gestion clavier complète.

Navigation clavier & ARIA

Skip Link

<a class="skip-link"
   href="#mainScroll">
  Aller au contenu principal
</a>

Zone d'annonce ARIA

<p id="zoneAnnouncer"
  aria-live="polite"
  aria-atomic="true"></p>

→ Annonce vocale du changement de zone au scroll

ProgressBar sémantique

<div role="progressbar"
 aria-valuemin="0"
 aria-valuemax="11000"
 aria-valuenow="0"
 aria-valuetext="0 mètre">

Sections nommées

Chaque bloc du HUD dispose d'un aria-labelledby ou aria-label. Les emojis décoratifs sont marqués aria-hidden="true".

Ce qui est implémenté

  • Piège à focus (Tab + Shift+Tab)
  • Fermeture à la touche Échap
  • Attribut inert sur le reste de la page
  • Retour du focus sur le bouton d'origine

ARIA de la dialog

role="dialog"
aria-modal="true"
aria-labelledby="overlayName"
aria-describedby="overlayMeta"
aria-hidden="true" ⟵ par défaut

Contrôles utilisateur

⚡ Réduction d'animations

  • Détecte prefers-reduced-motion
  • Peut être forcé manuellement
  • Canvas statiques si activé
  • GSAP ne lance pas ses animations
const isReducedMotion = () =>   forceReducedMotion ||
  motionQuery.matches;

🎨 Contraste renforcé

  • Ajoute .is-high-contrast sur body
  • Augmente les contrastes via CSS
  • Persisté en localStorage
<button
 aria-pressed="false"
 id="contrastToggle">
 Contraste renforcé: désactivé
</button>

Bilan accessibilité

✅ Points forts

  • Navigation 100% clavier
  • Lecteurs d'écran supportés (ARIA live)
  • Respect de prefers-reduced-motion
  • HTML sémantique (main, aside, section…)
  • lang="fr" sur <html>
  • Emojis aria-hidden="true"

⚠️ Améliorations possibles

  • Tester avec NVDA / VoiceOver
  • Vérifier ratios de contraste couleur
  • Alternative texte pour les canvas
  • Tests automatisés Lighthouse / axe
